# 外卖落下来啦！

手机优先的像素风外卖背包整理游戏网页版原型。

## 开始试玩

直接双击 `index.html`，或者在本目录启动本地服务：

```bash
python3 -m http.server 8000
```

然后打开 <http://localhost:8000>。

## 核心操作

- 手机游戏主界面固定为单屏：订单、天空、手持信息、背包和操作栏始终同时可见，不需要上下滚动。
- 外卖会沿三条轨道从天空持续坠落，点击包裹才能接住；不需要的可以放过。
- 接住后，背包会标出所有合法落点；点击绿色“+”即可放下，大件会自动贴齐格线。
- 手持食材时，能合成且有空间的食物显示绿色“合”，点击一次必定成功，不必先占空格。
- 中间手持栏会持续用大字显示拿到的食物、鲜度和完整合成式。
- 交付后，中间手持栏会明确保留“交付了什么、收入多少”的记录。
- 红色“!”表示配方正确但成品没有连续空间，提示会说明需要的形状和被什么挡住。
- 拖动食物可以整理位置；点选后也可旋转或拿到手上。
- 接取和合成会让其他食物鲜度减 1，移动、旋转、提交订单不消耗鲜度。
- 鲜度归零会变成差评块，并继承原食物占据的完整形状。
- 原料或低阶食物不能直接交高阶订单；高阶成品可以替代低阶订单。

## 当前配方

```text
面包 + 牛肉饼 → 汉堡排（1×2）
汉堡排 + 芝士片 → 芝士汉堡（L型）
芝士汉堡 + 可乐 → 芝士汉堡套餐（2×2）

炸鸡 + 炸鸡 → 炸鸡桶（2×2）
炸鸡桶 + 炸鸡桶 → 全鸡（2×2）

奶茶 + 奶茶 → 大杯奶茶（1×1）
大杯奶茶 + 奶油 → 奶盖奶茶（1×2）
```

## 试玩版功能

- 4×4 手动整理背包
- 1×1、1×2、2×2、L 型占格
- 实时空投、手持直合、背包内合成
- 鲜度、差评块、订单、高阶替代、升级与逐关难度
- 暖阳与赛博两套纯外观主题
- 本地进度保存和浏览器分享
- 桌面与手机响应式布局
